## 📌 Overview

This repository provides a **complete, structured cybersecurity learning roadmap** covering **12 core domains** — from foundational concepts to advanced topics like binary exploitation, APT intelligence, cloud security, and container hardening.

Built for **EWU Cybersecurity Club** members, junior researchers, CTF players, and anyone serious about mastering offensive & defensive security.

### 🎯 What You'll Learn

| Category | Topics |
|----------|--------|
| **Foundation** | Linux, Networking, Python, Security Mindset |
| **Offensive** | Web Attacks, PWN, Boot2Root, Active Recon, OSINT |
| **Defensive** | DFIR, Threat Hunting, SIEM, Blue Team Operations |
| **Cloud & Container** | AWS/Azure Security, Kubernetes Hardening, Container Escape |
| **Advanced** | APT Tradecraft, Red/Blue/Purple Team, Tor OSINT |

## 🚀 Getting Started

### Prerequisites

- A laptop/desktop with at least 8GB RAM (16GB recommended)
- VirtualBox or VMware for lab VMs
- Curiosity and persistence 🔥

### Recommended Learning Path
Foundation Zone → Core Domains → Web/OSINT → Active Recon → PWN → Red/Blue/Purple → APT → Cloud


### Tools You'll Master

| Category | Tools |
|----------|-------|
| 🔍 **Recon** | Nmap, masscan, Amass, Shodan, theHarvester |
| 💉 **Web** | Burp Suite, SQLmap, FFUF, Nuclei |
| 💣 **PWN** | pwntools, GDB/pwndbg, ROPgadget, Ghidra |
| 🔵 **Blue Team** | Splunk, Elastic, Zeek, Suricata, Velociraptor |
| 🔴 **Red Team** | Cobalt Strike, Mythic, Sliver, BloodHound |
| ☁️ **Cloud** | ScoutSuite, Pacu, kube-hunter, Trivy |
