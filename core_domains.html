👑 Privilege Escalation Deep Dive (Linux & Windows)

🐧 Linux Privesc
  • SUID binaries (GTFOBins)
  • Sudo misconfigurations
  • Cron jobs / wildcard injection
  • Kernel exploits (Dirty Pipe)
  • Capabilities & Docker groups
🪟 Windows Privesc
  • UAC bypass techniques
  • Token impersonation (JuicyPotato)
  • Service misconfigurations (Unquoted paths)
  • AlwaysInstallElevated
  • LSA secrets / SAM dump
📌 Post-Exploitation
  • Pivoting (ssh tunneling, chisel)
  • Data exfiltration (encrypted channels)
  • Persistence (scheduled tasks, services)
  • Lateral movement (PsExec, WMI)
