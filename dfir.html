📅 Timeline Analysis & Artifact Correlation
🔹 Acquisition (dd, E01) → 🔹 File carving (foremost) → 🔹 $MFT / Journal parse → 🔹 Plaso timeline → 🔹 Correlation (LNK, Prefetch, Shimcache)
📁 NTFS $MFT: timestamps, file creation, deletion 📋 Prefetch (Windows) — program execution history 🖥️ Shimcache — application compatibility evidence 📄 LNK files — recently accessed documents
Plaso (log2timeline) Timeline Explorer (SANS)
💡 Pro tip: Always work on a write-blocked forensic image. Hash verification (MD5/SHA256) ensures evidence integrity before analysis.
🧠 Memory Forensics with Volatility — Live Investigation
📋 Process Analysis
vol -f mem.raw windows.pslist → detect hidden processes (psscan, malfind)
🌐 Network Artifacts
windows.netscan → active connections, listening ports, injected DLLs
🔑 Registry from Memory
hivelist, hivedump — SAM, SYSTEM, SOFTWARE, userassist, run keys
🧬 Malware Detection
malfind (injection), yarascan, modscan (kernel modules), drivers
volatility3 -f dump.mem windows.psscan vol -f mem.raw windows.cmdline
🌐 PCAP Forensics Workflow
  • → tcpdump / tshark capture → analyze with Wireshark
  • → TCP stream reconstruction (Follow TCP Stream)
  • → TLS decryption using server private keys or session keys
  • → Zeek logs (conn.log, http.log, dns.log) for metadata
  • → Detect C2 beacons: periodic timing, JA3 fingerprints
tshark -r capture.pcap -Y "http.request" ngrep NetworkMiner
🔍 Zeek (Bro) — Network Meta-Forensics
  • 🔹 conn.log — connection summaries, duration, bytes
  • 🔹 http.log — URIs, user-agents, hosts
  • 🔹 dns.log — all queries, responses, TLD extraction
  • 🔹 files.log — extracted transferred files
  • 🔹 notice.log — anomaly detection (port scanning, C2)
zeek-cut RITA (Black Hills)

📦 Evidence Preservation — Order of Volatility
1. RAM / Registers — most volatile
2. Network state — connections, routing
3. Running processes — handles, DLLs
4. Disk (journal, swap) — slack space
5. Remote logs — SIEM, syslog
6. Backups / archives — least volatile
Write blocker (hardware) FTK Imager (E01) dcfldd (hash on the fly)
